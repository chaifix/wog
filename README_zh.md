# Wog    

[English](/README.md)   

![/doc/screenshot/a.gif](/doc/screenshot/a.gif)    

Wog是一个创建opengl窗口和处理输入的库。你可以吧 `wog.h` 和 `wog.c` 直接放到你的工程里使用。   

窗口应用和控制台应用都被支持了。如果要编译窗口引用，在 Visual Studio 中把子系统设置为 `Window` 。
如果要编译控制台应用，把子系统设置为 `Console`。
